# InfraPilot AI

An AI-powered Infrastructure Copilot designed to assist enterprise infrastructure teams with knowledge retrieval, troubleshooting, and incident reporting using Retrieval-Augmented Generation (RAG) and Agentic AI.

## Overview

Infrastructure engineers often spend valuable time searching through documentation, SOPs, and runbooks to resolve issues. InfraPilot AI aims to centralize this knowledge and provide intelligent assistance through a conversational interface.

## Objectives

- Answer infrastructure questions using company documentation.
- Provide context-aware troubleshooting assistance.
- Generate standardized incident reports.
- Improve knowledge sharing within the infrastructure team.

## Features (Planned)

- AI Knowledge Assistant (RAG)
- Diagnostics Agent
- Incident Report Agent
- User Authentication
- Chat History
- Document Management
- Role-Based Access Control

### Visual Knowledge Retrieval

When a retrieved SOP section contains screenshots,
InfraPilot AI displays the relevant screenshot
alongside the generated answer.

This allows engineers to verify procedures
using the original documentation.
